# ⚡ FPS Booster Dashboard (ChromeOS Friendly)

A **React-based performance simulator dashboard** that mimics FPS boosting, CPU optimization, and game-specific performance tuning. Designed for fun, UI experimentation, and ChromeOS-friendly browser use.

---

## 📁 Project Structure

```txt
FPS-Booster-Dashboard/
│
├── src/
│   ├── App.jsx
│   ├── hooks/useFpsCpu.js
│   ├── config/games.js
│   ├── config/tweaks.js
│   └── components/
│       ├── StatusCard.jsx
│       ├── GameOptimizer.jsx
│       └── TweaksCard.jsx
```

---

## 🚀 Features

* 📊  real-time FPS counter (requestAnimationFrame)
* 🧠 Simulated CPU usage monitor
* 🎮 Game optimizer presets (Fortnite, Valorant, Minecraft)
* ⚡ “Boost Performance” animation system
* 🧩 Modular React architecture (hooks, config, components)

---

## 🧠 How It Works

* FPS is simulated using browser frame timing
* CPU usage is randomly generated with optimization effects
* “Boost Performance” temporarily reduces simulated CPU load
* Game selection changes optimization tips dynamically

---

## 📦 Tech Stack

* React (Hooks)
* TailwindCSS
* ShadCN UI components
* requestAnimationFrame API
